<threat_model>
- T-14-03 Session fixation/expiry, severity medium: protected pages use `requireCustomerAccountSession()` and redirect expired sessions to login with safe return paths instead of rendering stale customer data.
- T-14-07 Customer data caching leak, severity high: all account route data is request-scoped, uncached, and rendered behind Suspense/loading shells rather than product/collection cache tags.
- T-14-02 Token exposure, severity high: Server Components pass only shaped display data to client leaves and Storybook fixtures use fake customer records only.
</threat_model>
