<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser prefetch to OAuth-start route | Next client prefetch could trigger a redirect chain to cross-origin Shopify OAuth endpoints |
| Local fake OAuth to live Shopify OAuth | Automated tests use fake/local endpoints; live OAuth requires admin-owned configuration |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-05 | Denial of Service / Information Disclosure | direct OAuth-start Next Link prefetch | mitigate | Add `prefetch={false}` to direct `/account/login/start` buttons only |
| T-15-06 | Spoofing / Misconfiguration | callback/logout origins | mitigate | Extend route tests and docs for configured-origin behavior; keep live OAuth owner-gated |
| T-15-OAUTH-SC | Information Disclosure | Customer Account tokens | avoid | Do not expose tokens or run live OAuth in automated tests |

</threat_model>
