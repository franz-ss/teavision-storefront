<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| First visitor page load to browser storage | Consent defaults must be denied before optional scripts can load |
| Consent UI to Shopify Customer Privacy API | Browser-only Shopify APIs may be unavailable or not configured |
| User preference state to analytics destinations | Analytics and marketing categories gate later destination dispatch |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-04 | Information Disclosure | consent defaults | mitigate | Default analytics and marketing to denied and test first-visit behavior |
| T-16-05 | Tampering | consent persistence | mitigate | Store only typed category values, reject malformed JSON, and preserve essential true |
| T-16-06 | Information Disclosure | Shopify privacy bridge | mitigate | Wrap Customer Privacy API in a typed browser-only adapter with unavailable/not-applicable status |

</threat_model>
