<threat_model>
Trust boundaries:

- Route metadata and sitemap output are consumed by crawlers before browser interaction.
- The launch host comes from environment variables and must not be confused with local/staging proof.

STRIDE register:

- T-18-05 spoofing: wrong canonical host or title suffix can misrepresent the launch URL/topic. Mitigation: keep SITE_URL env-driven and use `title.absolute` only on audit-target pages.
- T-18-06 tampering: blog tags can cannibalize main blog listing if indexed/sitemapped. Mitigation: tag metadata sets noindex and sitemap excludes tag URLs.
- T-18-07 information disclosure: robots/sitemap probes must not expose account/customer paths as indexable surfaces. Mitigation: disallow account/login surfaces and preserve `/search` noindex behavior.
  </threat_model>
