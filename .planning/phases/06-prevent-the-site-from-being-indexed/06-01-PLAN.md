<risks_and_edge_cases>

- Next metadata fields are shallowly merged by route segment; child `robots` metadata can overwrite the root robots directive unless local robots-bearing routes are wrapped.
- An empty sitemap should return a valid sitemap response while avoiding public storefront URLs.
- Static metadata may be evaluated at build time, so deployment must set `DISABLE_INDEXING=true` before building or deploying the gated version.
- Google must be able to crawl HTML pages to discover noindex, so robots.txt should not be turned into a blanket HTML crawl block as a substitute for metadata.
- Local product or blog response checks may require valid Shopify and Sanity credentials; if unavailable, record the blocker and still verify static routes, robots, sitemap, lint, and build behavior.

</risks_and_edge_cases>
