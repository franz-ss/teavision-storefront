<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser e2e to fake Shopify | Tests can mutate fake cart data but must not reach live checkout |
| Browser e2e to fake Customer Account | Tests can use fake sessions but must not run live OAuth without approval |
| Playwright webServer to local machine | Build/start lifecycle must be owned by the test command and cleaned up |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-07 | Tampering / Reliability | production e2e lifecycle | mitigate | Playwright starts fake providers and `next build`/`next start` itself |
| T-17-08 | Information Disclosure / Safety | checkout and OAuth tests | mitigate | Block third-party requests and assert no hosted checkout/OAuth navigation |

</threat_model>
