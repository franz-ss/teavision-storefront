# ComponentsSpecifications for the v1 component library. Every component here:- Consumes only **semantic** tokens (never primitives directly)- Is documented with **anatomy ‚Üí variants ‚Üí sizes ‚Üí states ‚Üí a11y ‚Üí usage**- Is organized so a designer can build the Figma component and an engineer can build the same component, and they'll match## Index1. **Layout** ‚Äî Container ¬∑ Section ¬∑ Divider2. **Typography utilities** ‚Äî Heading ¬∑ Eyebrow ¬∑ Link3. **Action** ‚Äî Button ¬∑ Icon Button4. **Form controls** ‚Äî Input ¬∑ Textarea ¬∑ Select ¬∑ Checkbox ¬∑ Radio Group ¬∑ Switch ¬∑ Form Field5. **Status & disclosure** ‚Äî Badge ¬∑ Tag / Filter Chip ¬∑ Certification Badge ¬∑ Tooltip ¬∑ Alert / Banner ¬∑ Toast ¬∑ Skeleton6. **Navigation** ‚Äî Header (with Mega Menu) ¬∑ Footer ¬∑ Breadcrumb ¬∑ Pagination ¬∑ Tabs7. **Cards** ‚Äî Card ¬∑ Product Card ¬∑ Article Card ¬∑ Testimonial Card ¬∑ Stat Block ¬∑ Catalogue Download Card8. **Patterns** ‚Äî FAQ Accordion ¬∑ Modal ¬∑ Drawer ¬∑ Quote Request Form ¬∑ Wholesale Account Form ¬∑ Spec Table ¬∑ Newsletter SignupTotal: **34 first-class components.**---## Universal rulesThese apply to every interactive component, so we don't repeat them per spec:- **Focus ring:** `box-shadow: var(--shadow-focus)` (3px amber-500 @ 35%) on `:focus-visible`. Never on `:focus` alone (avoids ring on click).- **Hit area:** minimum 40√ó40 px on touch devices. Visual size can be smaller; padding the hit area is fine.- **Disabled state:** `opacity: var(--opacity-40)`, `cursor: not-allowed`, `pointer-events: none` (or aria-disabled, depending on context).- **Loading state:** show inline spinner (icon-size matched to text), keep label visible, set `aria-busy="true"` on the trigger.- **Transition:** interactive state changes list the animated properties explicitly, usually `color`, `background-color`, `border-color`, `box-shadow`, `opacity`, and occasionally `transform`. Never use `transition: all`.- **Truncation:** single-line truncation uses `text-overflow: ellipsis`. Two-line truncation uses `-webkit-line-clamp: 2`.---## 1.
