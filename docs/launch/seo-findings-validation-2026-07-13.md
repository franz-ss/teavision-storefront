## Supporting evidence

### Crawlability and headings

- `/collections/all`: HTTP 200, about 450 KB HTML, one H1 (`All`), 27 unique product links in the response, product-card H3s present before browser JavaScript.
- `/collections/wholesale-bulk-tea`: HTTP 200, about 477 KB HTML, one H1 (`Wholesale Tea`), imported story headings exposed as H2.
- `/products/spearmint-organic`: HTTP 200, about 263 KB HTML, one product H1, accordion titles as H2, Product and BreadcrumbList JSON-LD.
- Targeted regression run: 4 test files and 28 tests passed for collection page content, product page headings, Shopify HTML normalization, and URL inventory.

### Metadata and schema spot checks

- All sampled staging pages use `lang="en-AU"` and `noindex, nofollow, noarchive`, as expected for protected staging.
- Staging canonicals point to `https://www.teavision.com.au`.
- Contact exposes LocalBusiness; supported service pages expose Service; FAQ/support pages expose FAQPage; PDP exposes Product.
- Product AggregateRating/Review was not emitted without trustworthy product review data.
